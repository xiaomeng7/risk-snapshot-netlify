# 网站测试指南

## 快速测试清单

### 1. 基础访问测试

- [ ] 网站可以正常打开
- [ ] 页面加载速度正常（< 3秒）
- [ ] 没有控制台错误（按 F12 查看）
- [ ] HTTPS 正常工作（地址栏显示锁图标）

### 2. 视觉和布局测试

- [ ] 页面布局正确
- [ ] 字体和颜色显示正常
- [ ] 深色/浅色主题自动切换（根据系统设置）
- [ ] 响应式设计：
  - [ ] 桌面端（> 1024px）
  - [ ] 平板端（768px - 1024px）
  - [ ] 手机端（< 768px）

### 3. 功能流程测试

#### 步骤 1：资格检查（Eligibility Check）

- [ ] 首页显示正确
- [ ] 点击 "Start (60 seconds) →" 按钮
- [ ] 问题 1：房产数量选择
- [ ] 问题 2：技术维护决策方式
- [ ] 问题 3：可用性描述
- [ ] 问题 4：是否愿意付费咨询
- [ ] 结果页面显示：
  - [ ] 如果不符合：显示 "Not suitable"
  - [ ] 如果符合：显示 "Suitable" 和 "Start Risk Snapshot" 按钮

#### 步骤 2：电气风险快照（Electrical Risk Snapshot）

- [ ] 快照介绍页面显示
- [ ] 点击 "Start Snapshot →"
- [ ] 完成 6 个快照问题
- [ ] 每个问题都能选择答案
- [ ] "Next" 按钮在未选择答案时禁用
- [ ] "Back" 按钮可以返回上一题
- [ ] 进度条实时更新

#### 步骤 3：结果页面

- [ ] 显示 "Snapshot complete"
- [ ] 显示不确定性评估结果
- [ ] "Book an independent assessment" 按钮可用
- [ ] "Prefer a quick call first?" 按钮可用
- [ ] "Learn more" 链接可用

#### 步骤 4：联系表单

- [ ] 点击 "Book an independent assessment"
- [ ] 表单字段显示：
  - [ ] 姓名（必填）
  - [ ] 电话（必填）
  - [ ] 邮箱（必填，需验证格式）
  - [ ] 备注（可选）
- [ ] 表单验证：
  - [ ] 空字段提交显示错误
  - [ ] 无效邮箱格式显示错误
  - [ ] 所有字段填写后可以提交

#### 步骤 5：成功页面

- [ ] 显示感谢信息
- [ ] 显示 "Send email to confirm" 链接
- [ ] 点击链接打开邮件客户端
- [ ] 邮件内容预填充正确

#### 步骤 6：快速通话模态框

- [ ] 点击 "Prefer a quick call first?"
- [ ] 模态框正确显示
- [ ] 表单字段完整：
  - [ ] 姓名、电话、邮箱
  - [ ] 房产区域（可选）
  - [ ] 日期选择器
  - [ ] 时间窗口选择
  - [ ] 备注（可选）
- [ ] 可以关闭模态框
- [ ] 表单验证正常工作

### 4. 导航测试

- [ ] "Back" 按钮可以返回
- [ ] "Restart" 按钮重置所有答案
- [ ] 进度条显示正确百分比
- [ ] 步骤标签（Step 1-4）正确高亮

### 5. 边界情况测试

- [ ] 刷新页面后状态保持（如果使用 localStorage）
- [ ] 浏览器前进/后退按钮
- [ ] 快速连续点击按钮不会出错
- [ ] 长文本输入不会破坏布局

### 6. 性能测试

- [ ] 页面加载时间 < 3秒
- [ ] 交互响应流畅（无卡顿）
- [ ] 图片/资源加载正常

### 7. 浏览器兼容性测试

在不同浏览器中测试：
- [ ] Chrome/Edge（最新版）
- [ ] Safari（最新版）
- [ ] Firefox（最新版）
- [ ] 移动浏览器（iOS Safari, Chrome Mobile）

### 8. 移动设备测试

- [ ] 触摸操作正常
- [ ] 表单输入正常
- [ ] 模态框在移动端显示正确
- [ ] 文本大小合适，无需缩放

## 常见问题检查

### 如果页面显示空白：
1. 检查浏览器控制台（F12）是否有错误
2. 检查网络请求是否成功
3. 确认文件路径正确

### 如果样式显示异常：
1. 清除浏览器缓存
2. 检查 CSS 文件是否加载
3. 检查是否有 CSS 冲突

### 如果表单提交失败：
1. 检查邮箱格式验证
2. 检查必填字段验证
3. 检查 mailto 链接是否正确

### 如果模态框不显示：
1. 检查 JavaScript 控制台错误
2. 检查模态框 HTML 是否存在
3. 检查 CSS 显示设置
